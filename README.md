# LVAgenticAI Final Project — Multi-Agent Research System

Multi-agent research/data-analysis system built on the Claude Agent SDK, exposed via FastAPI (WebSocket/SSE).

## Agents

| Name | Role |
|---|---|
| Hermione | Editor-in-Chief — plans, delegates, synthesizes the final report |
| Rita | Correspondent — web research (search + fetch) |
| Percy | Analyst — data analysis over fetched/synthetic data |
| Moody | Fact-Checker — adversarially verifies claims before synthesis |

See `changelog.md` for a per-file history of changes.
